# Hardware Design Licensing

Unless a file or subdirectory states otherwise, the hardware design materials
under this directory are licensed under the CERN Open Hardware Licence Version
2 - Strongly Reciprocal (`CERN-OHL-S-2.0`).

This scope includes hardware design sources and derived manufacturing artifacts,
including items such as schematics, PCB layouts, Gerbers, mechanical design
files, fabrication drawings, and bills of materials.

The full license text is available at:

- [../../LICENSES/CERN-OHL-S-2.0.txt](../../LICENSES/CERN-OHL-S-2.0.txt)

Notes for contributors:

- When possible, keep the preferred editable source form of a hardware design
  in the repository, not only exported PDFs, ZIP archives, or STL files.
- If the editable source is hosted elsewhere, document that source location in
  the relevant directory README or file notes.
- If you modify a hardware design and redistribute it, preserve notices and
  document the modification date and nature of the change, as required by
  `CERN-OHL-S-2.0`.

For software licensing, see [../../LICENSES/GPL-3.0.txt](../../LICENSES/GPL-3.0.txt).
