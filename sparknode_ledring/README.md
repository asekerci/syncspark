# SparkNode RGB LED Ring deBruijn Patterns Demo
## Overview

This program lights up the RGB LED ring of a sparknode with the
assigned deBruijn sequence. 

It is MQTT enabled. 

## ESP32-CAM and ESP-IDF Configuration

Configuration details (including the partition table) are in 
the main [README.md](../README.md) file of the **SyncSpark** project.

## Required Components

1. espressif/led_strip:  
 `idf.py add-dependency "espressif/led_strip^3.0.0"`
